Como colocar uma oficina no ar

Checklist manual (por enquanto) — vira automático quando ligarmos o provisionamento.

  1. Confirme os dados com o responsável pelo WhatsApp (botão na tabela).
  2. Receba logo, cores e catálogo de serviços da oficina.
  3. Personalize o ambiente (marca, domínio/subdomínio, serviços).
  4. Crie o primeiro admin da equipe no Supabase e faça um check-in de teste.
  5. Publique e mande os links (site, painel, app) para o responsável.

🚀 Quer isso multi-dispositivo e automático?

  • Hoje os cadastros ficam neste navegador (localStorage). Para vê-los de qualquer lugar, criamos a tabela leads no Supabase — eu já deixo o consultarPlaca/formulário gravando lá.
  • Provisionamento automático (cada oficina ganha o próprio ambiente/subdomínio) é a fase seguinte.
